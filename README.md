# JavelinThrowingVids
Dataset of javelin throwers performing the javelin throwing technique. The dataset has been categorized by the main javelin throwing fault identified in each video as well as what is an ideal technique.
Due to the restrictions in the size of the files that can be uploaded on github, the actual dataset can be found on Kaggle website. The link to the dataset is https://www.kaggle.com/zaavanrichards/javelin-throwing-videos
